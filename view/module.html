Read the short lesson below before starting the interactive game. Use the buttons on the top-right to switch the language.

ЁЯОп Learning Objective

Students will learn the main parts of a computer, common input & output devices, how a computer works, and basic terminology.

ЁЯЦея╕П What is a Computer?

A computer is an electronic device that takes data (Input), processes it (Processing), and gives results (Output).

ЁЯФ╣ Main Parts of a Computer

  • CPU тАУ Brain of the computer.
  • Monitor тАУ Shows results on the screen.
  • Keyboard тАУ Used to type letters and numbers.
  • Mouse тАУ Moves the pointer and selects items.
  • Speakers тАУ For listening to sound.
  • Printer тАУ Prints documents on paper.

ЁЯФ╣ Memory

  • RAM тАУ Temporary memory used while programs run.
  • Hard Disk / Storage тАУ Permanent storage for files and software.

ЁЯФ╣ Input & Output Devices

Input: Keyboard, Mouse, Scanner, Microphone

Output: Monitor, Speakers, Printer, Projector

ЁЯФ╣ Common Terms

  • USB тАУ Universal Serial Bus; used to connect devices (pen drive, mouse, keyboard).
  • Operating System (OS) тАУ Software that runs the computer (Windows, Linux, macOS).
  • Software тАУ Programs used for tasks (e.g., MS Word, Paint).
  • Hardware тАУ Physical parts of a computer.

ЁЯУЭ Summary

Computers work on the Input тЖТ Processing тЖТ Output principle. CPU, Monitor, Keyboard and Mouse are main parts. RAM is temporary memory and Hard Disk is permanent storage.
